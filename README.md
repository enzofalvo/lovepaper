# LovePaper 💌

> Plataforma de cartas de amor digitais com link permanente, música do Spotify e revelação agendada — em 5 idiomas.

**🔗 Live:** [lovepaper.app](https://lovepaper.app)
**💼 Modelo:** Freemium — template grátis (Minimal) + templates premium a partir de $2.99
**🌍 Idiomas:** PT · EN · ES · DE · FR

---

## 🎯 O problema

Casais em relacionamentos à distância, aniversários e datas especiais precisam de uma forma de tornar uma mensagem digital memorável — algo melhor que um WhatsApp ou um post no Instagram que some no feed.

## 💡 A solução

O usuário escreve a mensagem, escolhe um template, personaliza, e em menos de 5 minutos recebe uma página romântica com link único e permanente. Pode agendar a abertura para uma data futura e adicionar música do Spotify/YouTube.

## 🏗️ Stack & arquitetura

- **Frontend:** Next.js (React + TypeScript) com SSR para SEO
- **i18n:** internacionalização completa para 5 idiomas
- **Integrações:** Spotify API · YouTube embeds
- **Pagamentos:** gateway internacional (compra única por template premium)
- **SEO:** structured data, sitemap multilíngue, OG dinâmico por carta
- **Persistência:** banco para cartas + sistema de links permanentes

### Desafios técnicos resolvidos
- Geração de página dinâmica única e permanente por carta criada
- Sistema de agendamento de "abertura" com criptografia até a data marcada
- SEO multilíngue ranqueando em 5 mercados
- Otimização de performance para Core Web Vitals (página carregada em < 1s)

## 📊 Status
Produto em produção. Usuários pagantes recorrentes em múltiplos países.
